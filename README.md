## 🎯 Project Overview

This comprehensive research project develops an integrated IoT and AI-powered ecosystem that revolutionizes mushroom cultivation practices. Our framework combines real-time environmental monitoring, computer vision-based analysis, and predictive analytics to optimize every aspect of mushroom farming—from initial planning to harvest optimization.

### 🌟 Vision Statement
> *"Empowering mushroom growers with intelligent technology to maximize yield, minimize losses, and ensure sustainable cultivation practices through data-driven decision making."*

## 🚀 Quick Start

### Prerequisites

Ensure you have the following installed on your system:

```bash
# Python 3.8 or higher
python --version

# pip package manager
pip --version

# Git version control
git --version
```

### ⚡ Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/abheethasma/Research-Project-Of-Mushroom.git
   cd Research-Project-Of-Mushroom
   ```

2. **Set Up Virtual Environment**
   ```bash
   python -m venv mushroom_env
   source mushroom_env/bin/activate  # On Windows: mushroom_env\Scripts\activate
   ```

3. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Configure Environment Variables**
   ```bash
   cp .env.example .env
   # Edit .env with your configuration
   ```

5. **Initialize Database**
   ```bash
   python scripts/init_db.py
   ```

### 🎮 Running the Application

```bash
# Start the backend server
python src/app.py

# In a new terminal, start the frontend
cd frontend
npm start

# For IoT components
python src/iot/sensor_monitor.py
```
