### Prerequisites

Ensure you have the following installed on your system:

```bash
# Python 3.8 or higher
python --version

# pip package manager
pip --version

# Git version control
git --version
```

### âš¡ Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/abheethasma/Research-Project-Of-Mushroom.git
   cd Research-Project-Of-Mushroom
   ```

2. **Set Up Virtual Environment**
   ```bash
   python -m venv mushroom_env
   source mushroom_env/bin/activate  # On Windows: mushroom_env\Scripts\activate
   ```

3. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Configure Environment Variables**
   ```bash
   cp .env.example .env
   # Edit .env with your configuration
   ```

5. **Initialize Database**
   ```bash
   python scripts/init_db.py
   ```

### ğŸ® Running the Application

```bash
# Start the backend server
python src/app.py

# In a new terminal, start the frontend

npm start

# For IoT components
python src/iot/sensor_monitor.py
```
